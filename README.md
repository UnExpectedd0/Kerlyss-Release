**Kerlyss** is a high-performance, unified audio ecosystem designed to bridge the gap between local high-fidelity libraries and global streaming services. This repository serves as the official distribution channel for production-ready binaries.

---

## ✨ Key Features

*   **Unified Library:** Seamlessly browse and play your local files alongside YouTube and streaming content.
*   **Aether Design System:** A stunning, state-of-the-art glassmorphic UI built for speed and aesthetics.
*   **Cross-Platform:** High-performance builds for Android and Windows environments.
*   **Smart Discovery:** Integrated search and discovery across multiple global providers.
*   **Auto-Update Ready:** Built-in update notification system to keep you on the latest version.

## 📥 Installation

### 🤖 Android
1. Go to the [Latest Release](https://github.com/UnExpectedd0/Kerlyss-Release/releases/latest).
2. Download the `Kerlyss-Android.apk`.
3. Open the file on your device and follow the installation prompts.

### 🪟 Windows
1. Go to the [Latest Release](https://github.com/UnExpectedd0/Kerlyss-Release/releases/latest).
2. Download the `Kerlyss-Windows.zip`.
3. Extract the folder to your preferred location.
4. Run `kerlyss.exe` to start the application.

## 🛡️ Security & Integrity

We take security seriously. Every release includes a `checksums.txt` file containing SHA-256 hashes. You can verify your download using the following commands:

**Windows (PowerShell):**
```powershell
Get-FileHash .\Kerlyss-Windows.zip -Algorithm SHA256
```

**Android/Linux/macOS (Terminal):**
```bash
sha256sum Kerlyss-Android.apk
```
